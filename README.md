# 🛠️ GodMode Pro – Windows God Mode Manager  
---

## 📌 Overview  
**GodMode Pro** is a lightweight Windows batch-based utility designed to enable and disable the hidden **God Mode** feature in a clean and controlled way.  

It provides a simple, menu-driven interface to access advanced system settings without manual steps.

---

## ✅ Key Highlights  
✔ Works on Windows 10 & 11  
✔ No installation required  
✔ Administrator access required  
✔ Enable & disable God Mode instantly  
✔ Clean, minimal, and reliable script  

---

## 🚀 Features  

| Option | Function |
|--------|---------|
| 1 | Enable God Mode (Creates special folder on Desktop) |
| 2 | Disable God Mode (Removes God Mode folder) |
| 3 | Exit Tool |

---

## 🧠 What is God Mode?  
**God Mode** is a special Windows feature that provides access to **all Control Panel settings in one place** using a hidden CLSID.  

It allows users to quickly manage system configurations without navigating through multiple menus.

---

## 📥 Usage  
1. Download the script
2. Right-click → Run as Administrator
3. Choose an option from the menu
